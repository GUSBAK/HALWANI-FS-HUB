# Halwani Food Service CRM v3

Clean Next.js app for Vercel.

## Vercel settings
- Framework: Next.js
- Build command: npm run build
- Output directory: leave blank
- Root directory: .

## Local run
npm install
npm run dev

## Current modules
- Login screen
- Mobile dashboard
- Today's journey
- Customer snapshot
- GPS check-in and check-out, 20 meter rule
- Visit timer
- Visit recording
- Competitor updates
- Action list
- CSV export with Google Maps links

## Notes
This version uses demo data in `lib/demoData.js`. Supabase client is ready in `lib/supabaseClient.js` for the next phase.
